Random Forest

Ensemble learning model based on multiple decision trees for pest alert prediction.

Step 01

Dataset Split

The dataset was divided into 80% training data and 20% testing data using train_test_split() with random_state=42.

Step 02

Training Process

Random Forest trained multiple decision trees using climate and agricultural variables to improve prediction stability.

Step 03

Hyperparameters

The model used 200 trees, max_depth=10, class_weight='balanced', and random_state=42.

Code Section Purpose
train_test_split() Separates training and testing data
RandomForestClassifier() Creates ensemble classification model
model.fit() Trains the model using historical data
model.predict() Generates pest alert predictions

Flask Integration

Integration of trained machine learning models into the Flask web application for real-time agricultural pest alert prediction.

Prediction Workflow

  • Flask receives agricultural and climate information submitted by the user through the prediction form.
  • Input variables such as crop type, municipality, temperature, and precipitation are transformed into numerical representations.
  • The processed information is sent to the trained Random Forest model.
  • The machine learning model generates a pest alert prediction using learned agricultural patterns.
  • The final prediction result is displayed dynamically in the web interface.

Prediction Code Example 

input_data = pd.DataFrame([user_data])

prediction = model.predict(input_data)

result = (
    "High Pest Risk"
    if prediction[0] == 1
    else "Low Pest Risk"
)

Flask Route Integration

Backend route used to connect the prediction form with the trained machine learning model. 

@app.route("/prediction", methods=["POST"])

def prediction():

    user_data = request.form

    prediction = model.predict(input_data)

    return render_template(
        "prediction.html",
        prediction=prediction
    )
