Technical justification for using Logistic Regression as the baseline model.

Logistic Regression as the reference model

Logistic Regression offers strong interpretability, efficient training, good classification performance, and stable cross-validation results. If the difference versus Random Forest and Gradient Boosting is small, it remains a very strong choice due to simplicity and lower overfitting risk.

Technical justification

  • Direct interpretability through coefficients.
  • L2 regularization with C=1.0.
  • Class balancing with class_weight='balanced'.
  • Fast and stable training process.
  • Good generalization with 5-fold cross-validation.
