Code Explanation

Short summary of what the code does.

  • Data is loaded and cleaned from processed_alerts.csv.
  • Dataset split: 80% training / 20% test with stratification.
  • Preprocessing: StandardScaler is applied.
  • Configuration: C=1.0, solver=lbfgs, max_iter=500, class_weight=balanced, random_state=42.
  • Training: clf.fit(X_train_s, y_train).
  • 5-fold cross-validation with cross_val_score.
  • Prediction on the test set.
  • Coefficients are extracted to explain the model.
