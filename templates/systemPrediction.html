Business Understanding

Agricultural decision-support system for early pest risk detection.

Problem

Farmers require faster identification of environmental conditions associated with pest propagation.

Goal

Transform machine learning predictions into a practical web application for agricultural monitoring.

Users

Agricultural technicians, researchers, and crop monitoring teams.

Impact

Supports preventive actions and reduces economic losses caused by pest outbreaks.

Model Engineering

Random Forest combines multiple decision trees to improve classification stability.

Selected Features

Temperature, precipitation, municipality, and relative crop yield are used as prediction variables.

Training Strategy

The model was trained using supervised learning and evaluated using classification metrics.

Prediction Logic

The system aggregates multiple decision trees to produce a final pest alert classification.

Flask Integration

User input is processed in the backend and passed directly into the trained model.
